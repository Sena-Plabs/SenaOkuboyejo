## Quick start
1. Create a repo named `<your-username>.github.io`.
2. Upload everything from this zip to the repo root.
3. Replace **`<your-username>`** in: `index.html`, pages, `sitemap.xml`, `robots.txt`.
4. (Optional) If you have a custom domain, create a `CNAME` file at the repo root with just your domain.
5. Replace `/docs/Sena-Ok-CV.pdf` with your official CV if desired.

## Projects feed
Edit `/assets/data/projects.json` and push — cards update automatically on Home and /case-studies/.
