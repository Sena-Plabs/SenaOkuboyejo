Published at: **https://sena-plabs.github.io/SenaOkuboyejo/**

**Quick publish**
1. Create repo `senaokuboyejo` in the `sena-plabs` account.
2. Upload all files to the repo root.
3. In **Settings → Pages**, select **main** branch, `/ (root)`.
4. Replace `docs/Sena-Ok-CV.pdf` with your official CV anytime.
5. Add/modify projects in `/senaokuboyejo/assets/data/projects.json`.

All links are pre-scoped to `/senaokuboyejo` for project-site hosting.
